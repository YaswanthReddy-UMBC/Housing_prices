# Housing_prices
This small project is basically to predict the pricing of houses based on their location, income, proximity to school and other facilities etc.
It has following features: ['School_Rating', 'House_Area', 'Lot_Area',"Room_No","Bath_No","Garage","Pool","House_Age","Walkability","Crime_Rate","Zipcode"]
Step 1:
Upload the CSV file in google colab.
Step 2:
Performing Data cleaning in colab.
Step 3:
Splitting the data set in to training and test data to train and test the model.
Step 4:
Building a linear regression model and validating it.
Step 5:
Validating the model by giving external data.
